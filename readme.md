## 修改配置

1. `mysql/connect_mysql.js` 中修改数据库用户名密码
2. 新建数据库 `chat`, 导入`DB`中的`chat.sql`
3. 进入数据库 `chat`, 导入`DB`中的`songs.sql`

## 运行

```shell
npm install
node app
```
